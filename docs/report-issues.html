Before Reporting

신고 전 확인사항

문제를 신고하기 전에 아래 내용을 확인하면 더 빠른 해결이 가능합니다.

에러 로그 확인 및 복사

설정(⚙) → 맨 아래 "로그" 버튼 → "📋 복사" 버튼을 클릭하여 에러 로그를 복사합니다. 신고 양식에 붙여넣으면 원인 파악이 훨씬 빨라집니다.

증상 정리

어떤 상황에서 문제가 발생하는지, 어떤 동작을 했을 때 발생하는지, 항상 발생하는지 가끔 발생하는지 등을 정리해두세요.

버전 확인

사이드 패널 상단 또는 설정 → "걸러드림에 대하여..."에서 현재 버전을 확인합니다.

💡 팁: 로그 복사 방법을 모르겠다면 사용 가이드 → 에러 로그 섹션을 참고하세요.
