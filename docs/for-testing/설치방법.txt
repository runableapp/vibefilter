걸러드림 (VibeFilter) v3.0.0
https://runableapp.github.io/vibefilter/
================================================



■ 이게 뭐하는 거야?
================================================
한국 주요 커뮤니티 15곳(클리앙, 펨코, 루리웹, 더쿠, 뽐뿌, 보배드림,
82쿡, 오늘의유머, MLB파크, SLR클럽, 인벤, 인스티즈, 이토랜드,
웃긴대학, 와고)의 인기글을 자동으로 모아서, 네가 읽을 만한 글만
골라주는 Chrome 확장 프로그램이야.

- 서버 없음: 모든 처리가 네 브라우저 안에서만 이루어짐
- AI 학습: 네가 읽는 글/안 읽는 글을 학습해서 추천 정확도가 점점 올라감
- 무료: 돈 안 듦, 개인정보 수집도 없음




■ 설치 방법
================================================

1. 압축 파일 풀기
   - vibefilter-3.0.0.zip 파일을 원하는 폴더에 압축 해제해.
   - 압축 해제한 폴더 위치를 기억해 둬.

2. Chrome 확장프로그램 페이지 열기
   - Chrome 주소창에 chrome://extensions 입력 후 Enter.
   - 또는: 메뉴(⋮) > Extensions > Manage Extensions

3. 개발자 모드 켜기
   - 페이지 오른쪽 상단의 "Developer mode" 토글을 켜.

4. 확장프로그램 로드
   - 왼쪽 상단 "Load unpacked" 버튼 클릭.
   - 1단계에서 압축 해제한 폴더를 선택해.
     (manifest.json 파일이 있는 폴더)

5. 설치 확인
   - 목록에 "걸러드림"이 나타나면 완료!
   - 브라우저 오른쪽 상단 퍼즐 아이콘(Extensions)을 눌러서
     걸러드림을 핀 고정하면 편해.




■ 초기 설정
================================================

1. 사이드 패널 열기
   - 툴바의 걸러드림 아이콘을 클릭하면 팝업이 뜨는데,
     "사이드 패널" 버튼을 누르면 오른쪽에 글 목록 패널이 열려.

2. 설정 페이지 열기
   - 팝업에서 "설정" 버튼을 누르거나,
   - 사이드 패널 상단 ⚙ 아이콘을 눌러.

3. 커뮤니티 사이트 선택
   - 설정 > "커뮤니티 사이트" 에서 보고 싶은 사이트만 켜/꺼.
   - 꺼진 사이트의 글은 목록에서 안 나와.

4. AI 임베딩 모델 설정
   - 설정 > "🤖 AI 임베딩" 섹션으로 가.
   - "임베딩 활성화" 버튼을 누르면 모델 다운로드가 시작돼.
   - 약 120MB 정도 다운로드하는데, 처음 한 번만 받으면
     이후에는 브라우저에 캐시됨.
   - 다운로드 진행률이 표시되고, 완료되면 "준비 완료"로 바뀜.
   - 이걸 켜면 추천 정확도가 눈에 띄게 올라가고,
     토픽 필터도 더 정확해져.

5. 토픽 필터 (선택사항)
   - 설정 > "토픽 필터"에서 보기 싫은 주제를 체크하면
     해당 주제의 글이 자동으로 숨겨져.

6. 키워드 필터 (선택사항)
   - 설정 > "키워드 필터"에서 특정 단어가 제목에 포함된
     글을 숨길 수 있어.

7. 설정 저장
   - 설정을 바꾼 후 맨 아래 "설정 저장" 버튼을 꼭 눌러!


■ 사용 방법
================================================

1. 글 목록 보기
   - 사이드 패널을 열면 각 커뮤니티의 인기글이 리스트로 나와.
   - 30분마다 자동으로 새 글을 수집해.
   - 수동으로 새로고침하려면 사이드 패널 상단의 새로고침 버튼.

2. 글 읽기
   - 리스트에서 글 제목을 클릭하면 해당 사이트로 이동해.
   - 읽은 글은 "좋아하는 글"로 AI가 학습함.

3. 관심 없는 글 거부
   - 글 옆의 ✕ 버튼을 누르면 "관심 없음"으로 처리돼.
   - AI가 이걸 참고해서 비슷한 글의 추천 점수를 낮춰.

4. 작성자 차단
   - 특정 작성자의 글을 계속 보기 싫으면 차단할 수 있어.
   - 차단된 작성자의 글은 목록에서 사라지고, 싫어하는 글로 학습돼.

5. AI 추천 점수
   - 30개 이상의 글을 읽으면 AI가 추천 점수를 매기기 시작해.
   - 점수가 높은 글은 위쪽에 표시되고, 추천 마크가 붙어.
   - 많이 쓸수록 정확도가 올라가!


■ 참고사항
================================================
- "Developer mode"는 항상 켜져 있어야 해.
- Chrome 시작 시 개발자 모드 확장프로그램 관련 알림이
  나올 수 있는데, 무시해도 돼.
- 업데이트가 필요하면 기존 걸 "Remove"하고 새 파일로 다시 설치.
- 모든 데이터는 네 브라우저 안에만 저장돼. 서버 전송 없음.
